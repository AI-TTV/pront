プロンプト合体ツール README
━━━━━━━━━━━━━━━━━━━━━━

このツールは、
「固定プロンプト」と「シーンプロンプト」を
最短・最小ストレスで合体し、コピーするためのツールです。

インストール不要・ブラウザで動作します。

──────────────────────
【1】使い方（最短ルート）
──────────────────────

1. ZIPを解凍してください
2. 中の「index.html」をダブルクリックで開きます
3. 左でシーンを選ぶ（または貼る）
4. 右の「合体してコピー」を押す
5. そのまま貼り付けて使えます

※ 固定プロンプトは、画面上部の
　「固定プリセット」から切り替えできます。

──────────────────────
【2】コピーできない場合（重要）
──────────────────────

ごく一部の環境では、
自動コピーが制限されることがあります。

その場合でも、必ずコピーできます。

▼ 手動コピー手順
1. 「合体結果を開く」を押す
2. 表示されたテキストをクリック
3. Ctrl + A（全選択）
4. Ctrl + C（コピー）

※ これが最終手段で、100%確実です。

──────────────────────
【3】保存について
──────────────────────

・シーン
・固定プロンプト

どちらも、保存・呼び出しが可能です。
編集途中の内容は自動で下書き保存されます。

ただし、以下の環境では保存が効かない場合があります：
・シークレットモード
・会社PCなどの制限環境

──────────────────────
【4】保存できない環境での対処法
──────────────────────

ツール右下の「エクスポート」を使ってください。

1. 「エクスポート（JSON）」を押す
2. ファイルとして保存
3. 別の環境では「インポート」で復元

これで、どんな環境でも運用できます。

──────────────────────
【5】おすすめの使い方
──────────────────────

・普段は Web版（URL）を使用
・オフライン時や制限環境では ZIP版を使用
・重要なデータは定期的にエクスポート

──────────────────────
【6】注意事項
──────────────────────

・本ツールはローカル環境で動作します
・入力データは外部に送信されません
・自己責任でご利用ください

※保存データはブラウザ内に保存されます。
ブラウザの「サイトデータ削除」「履歴の完全削除」等を行うと消えるため、
必要に応じて「エクスポート（JSON）」でバックアップしてください。
